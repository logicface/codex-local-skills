# Research Writing Polish

Use this skill for publication-oriented polishing of academic prose.

## Routing boundary

- Use this skill for general journal, conference, thesis, and report polishing.
- If the user explicitly requests `Nature`, `Nature style`, high-impact-journal house style, British English, strict section-aware tense/hedging control, or overclaim auditing, prefer `nature-polishing`.

## English LaTeX mode

- Improve grammar, sentence structure, clarity, and academic tone.
- Keep LaTeX commands, equations, and existing technical abbreviations intact.
- Prefer simple, standard scientific wording over flashy vocabulary.
- Do not add new formatting or list structure unless the user asks.

## Chinese paper mode

- Fix only necessary issues: grammar, obvious logic gaps, unclear references, or spoken-style phrasing.
- If the original Chinese is already publication-ready, preserve it rather than forcing variation.
- Keep output Word-friendly plain text with standard full-width punctuation.

## General rules

- Preserve author intent and technical meaning.
- Avoid over-editing for cosmetic variety.
- If useful, provide a short change summary rather than long commentary.
