# Research Writing Redline Review

Use this skill for final-pass review and high-severity critique.

## Core rules

- Default to high tolerance. Do not nitpick style if the text is already acceptable.
- Report only substantive issues: logical contradictions, terminology drift, unclear claims, unsupported leaps, or serious grammar that blocks understanding.
- If there are no material issues, say so explicitly.
- Distinguish reviewer-facing concerns from mere rewriting preferences.

## Use cases

- Final consistency and logic check for a section
- Reviewer-style audit of a full manuscript or PDF
- Red-flag screening before submission

## Output expectations

- If clean: provide a short pass result.
- If problematic: list only the material issues, ordered by severity.
- For whole-manuscript review, separate contribution concerns, method concerns, experiment concerns, and clarity concerns.
