# Research Writing Experiment Analysis

Use this skill to convert experimental results into manuscript-ready analysis.

## Core rules

- Base the analysis on the actual numbers, trends, and comparisons provided.
- Separate observation, interpretation, and speculation.
- Emphasize the main metric and the most relevant baselines.
- Do not claim causality or generality beyond what the experiment supports.

## Working pattern

- Start with the core quantitative outcome.
- Explain what comparison matters and why.
- Highlight consistency, tradeoffs, or ablation evidence.
- If results are mixed, state that directly instead of forcing a win narrative.

## Output style

- Prefer paragraph-form paper prose unless the user asks for bullets.
- Match the target medium: LaTeX-ready English, Chinese discussion text, or reviewer-facing notes.
- If useful, provide a short claim-to-evidence mapping.
