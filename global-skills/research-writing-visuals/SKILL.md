# Research Writing Visuals

Use this skill for paper figure planning and caption/title generation.

## Routing boundary

- Use this skill for chart choice, panel logic, titles, captions, and figure planning.
- If the user explicitly wants a `Nature` or high-impact-journal figure workflow, actual plotting scripts, backend choice, export packages, or production QA, prefer `nature-figure`.

## Scope

- Method or architecture figure planning
- Experimental chart recommendation
- Figure title or caption generation
- Table title or caption generation

## Core rules

- Optimize for clarity, academic aesthetics, and evidential value.
- Match the visual choice to the data type and claim being supported.
- Keep labels concise and publication-appropriate.
- Separate what should be shown in the figure from what should stay in the main text.

## Working pattern

- For architecture figures, identify modules, data flow, novelty, and what text labels are actually necessary.
- For experiment plots, choose from standard academic chart types based on variable type, comparison structure, imbalance, uncertainty, and tradeoff goals.
- For figure or table titles, write concise, descriptive, non-hyped titles that tell the reader what evidence is being shown.
