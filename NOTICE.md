# Notice

This is a public aggregation repository that packages skill folders exported from a local Codex environment.

Important constraints:

- The repository contains mixed-origin content.
- Some directories include their own `LICENSE`, `README`, plugin metadata, or asset bundles.
- Exact upstream commit hashes were not preserved during the export process.
- No repository-wide blanket license is declared at the root at this stage.

Before reusing or redistributing any individual skill directory:

1. Check that directory for its own license or attribution files.
2. Review `SOURCES.md` for the current best-effort provenance mapping.
3. Treat project-specific skills as repository-coupled workflow artifacts rather than general-purpose public libraries unless you review them first.
